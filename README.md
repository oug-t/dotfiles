## Management

Managed entirely with [GNU Stow](https://www.gnu.org/software/stow/).

```bash
git clone [https://github.com/yourusername/dotfiles.git](https://github.com/yourusername/dotfiles.git) ~/dotfiles
cd ~/dotfiles
stow <package_name>
```
